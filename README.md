
# 야구선수 관리 프로그램

## 주요 기능
- 선수 등록, 선수 전체 목록
- 팀 등록, 팀 전체 목록
- 팀별 선수 목록, 선수 포지션별 목록
- 퇴출 선수 등록, 퇴출 선수 목록
- 야구장 등록, 전체 야구장 목록

## 역할
- 언약: 팀 등록, 팀 전체 목록, 팀별 선수 목록, 야구장 등록, 야구장 전체 목록 

- 하얀: 선수 등록, 선수 전체 목록, 퇴출 선수 등록, 퇴출 선수 목록, 선수 포지션별 목록

## 사용한 기술

### 기술 스택
- JDK 11 
- MySQL 8.0
### 의존성
- lombok
- MySQL connector
